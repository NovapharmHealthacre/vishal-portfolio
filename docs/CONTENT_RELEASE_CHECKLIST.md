# Founder content release checklist

Use before any merge from `content/founder-authority-rewrite`.

- [ ] Public copy is founder-led and pharmaceutical-specific.
- [ ] No unsupported licence, customer, revenue, product approval, partnership or deployed-technology claim exists.
- [ ] Retired essay title and narrative are absent from public output.
- [ ] Previous essay URLs resolve to neutral `noindex,follow` compatibility pages.
- [ ] At least six new cornerstone essays are public.
- [ ] Private drafts remain excluded from output.
- [ ] Titles, descriptions, H1s and summaries are unique.
- [ ] Technical essays use appropriate primary sources.
- [ ] Article dates reflect actual publication and material review.
- [ ] `facts.json`, JSON-LD, sitemap, RSS, JSON Feed and `llms.txt` remain valid.
- [ ] `/facts.json` is not promoted in the visible interface.
- [ ] Navigation displays `Profile` while `/facts/` remains stable.
- [ ] No tracking or third-party runtime script was added.
- [ ] Existing visual design, portrait and canvas behaviour remain intact.
- [ ] `npm ci` succeeds.
- [ ] `npm run check` succeeds.
- [ ] Hosted CI succeeds on the exact PR head.
- [ ] Desktop and mobile rendered output is reviewed.
- [ ] Accessibility and performance do not regress.
- [ ] Owner reviews all public essays and page copy.
