# Measurement specification

Reviewed: 14 July 2026

## Current privacy position

The website uses no analytics, advertising trackers, contact-form provider or non-essential cookies. This phase preserves that position.

## Approved measurement sources

| Source | Measures | Data location | Website change required |
|---|---|---|---|
| Google Search Console | Queries, impressions, clicks, pages, countries, devices, indexing, HTTPS and Core Web Vitals | Google account | No runtime tracking added |
| Bing Webmaster Tools | Search performance, crawl, index coverage, backlinks and optional IndexNow reporting | Microsoft account | No runtime tracking added |
| GitHub Actions | Build, test and deployment success | GitHub repository | Existing workflows only |
| Public search audits | Titles, snippets, images, panels and stale results | Manual observation | None |
| LinkedIn analytics | Post and profile distribution performance | LinkedIn account | None |
| Email enquiry log | Qualified enquiries, subject and source where voluntarily stated | Owner’s business email process | No website database |

## Core measures

### Search discovery

- indexed canonical pages;
- branded query impressions;
- non-branded topic queries;
- clicks and click-through rate;
- page-level search visibility;
- image-search visibility;
- crawl/index errors;
- stale legacy result disappearance.

### Authority

- verified external mentions;
- publisher-hosted contributions;
- qualified referring domains;
- interview/editorial invitations;
- citations of original frameworks;
- accurate entity associations.

### Content

- essay impressions and clicks;
- LinkedIn distribution performance;
- substantive comments and professional discussions;
- related enquiries;
- source/citation health;
- material update cadence.

### Reliability

- production uptime checks;
- HTTPS and custom domain;
- deployment success;
- Lighthouse regression evidence during releases;
- Search Console field Core Web Vitals where available.

## Excluded tools

Do not install Google Analytics, Meta Pixel, LinkedIn Insight Tag, Hotjar, session replay, fingerprinting, advertising pixels, marketing automation or cookie-consent tooling without a separate approved privacy and performance review.

## Future analytics decision template

Any proposal must state:

- business question;
- exact data collected;
- processor and data location;
- cookies/local storage;
- retention;
- consent and legal implications;
- privacy-page changes;
- performance cost;
- security and access controls;
- owner approval.

## Reporting cadence

- Weekly for the first month after an approved SEO merge.
- Monthly for search, content and external authority.
- Quarterly for facts, entity consistency and crawler policy.

Private console exports and email content must remain outside the public repository.
